## 프로젝트의 실행 방법
1. 레포지토리 복사:
```
https://github.com/lyn94/wanted-pre-onboarding-frontend.git
```
2. 경로 이동:
```
cd wanted-pre-onboarding-frontend
```
3. 설치:
```
npm install
```
4. 프로젝트 실행:
```
npm start
```
